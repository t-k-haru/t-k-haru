### Featured project: CodeBridge

An AI agent that redesigns Japan's ringi approval workflow. From a natural-language request it drafts the change, runs a sandbox self-debug loop, auto-classifies the approval type, recommends the right approver, and logs every decision for audit. About 80 percent of approvals are confirmation-type the AI can prepare, so people focus on real judgment. AI proposes; a human always approves before anything ships.

Stack: FastAPI / Python, o4-mini LLM, JWT + SQLite, Docker / Render
Repo: https://github.com/t-k-haru/CodeBridge
Live demo: https://codebridge-ringi.onrender.com
